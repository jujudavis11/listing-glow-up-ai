# Listing Glow Up AI MVP

A polished Next.js MVP web app for realtors to generate listing marketing assets from one photo and quick property details.

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start dev server:
   ```bash
   npm run dev
   ```
3. Open http://localhost:3000

## Notes

- Uses local state only (no DB/auth/billing).
- Outputs are template-generated for MVP.
- Includes TODO markers for OpenAI API integration and PNG export wiring.
